## 什麼是 Microsoft MakeCode？

Microsoft MakeCode 是 micro:bit 的預設程式編輯介面，是由微軟 Microsoft 開發的一款圖形化程式開發介面，其中包括有趣專案、即時結果，以及適合不同學員程度的積木編輯器和文字編輯器，為所有學生開啟電腦科學學習之路。
